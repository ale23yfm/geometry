Chapter 1 — Affine Space & Coordinates

Affine Space Basics

An affine space A = set of points + vector space V = D(A) with maps:

□ − □ : A × A → V (two points give a vector) □ + □ : V × A → A (vector moves a point) Axioms: 0+P=P, (v+w)+P=v+(w+P), (A−B)+B=A, (v+P)−P=v
Affine subspace of dimension d
S = {Q + t₁v₁ + ··· + t_d v_d : tᵢ ∈ ℝ} D(S) = direction space (d-dim vector subspace) • dim 0 = point, dim 1 = line • dim 2 = plane, dim n−1 = hyperplane
Parallel subspaces
S ∥ T ⟺ D(S) ⊆ D(T) or D(T) ⊆ D(S) Through any point Q ∃! subspace ∥ S with same dim
Dimension of intersection (Grassmann)
dim(S)+dim(T)−n ≤ dim(S∩T) ≤ min{dim S, dim T} Equality on left ⟺ D(S)+D(T) = Vₙ

Cartesian Frames & Coordinates

Frame K = (O, B), basis B = (i₁,…,iₙ)
P(x₁,…,xₙ): OP = x₁i₁ + ··· + xₙiₙ [P]_K = column matrix of coordinates
Change of frame K→K′ = (O′, B′)
[P]_{K′} = M_{B′,B} · ([P]_K − [O′]_K) Steps: 1. Build M_{B,B′}: columns = [i′ⱼ]_B 2. Invert: M_{B′,B} = (M_{B,B′})⁻¹ 3. Apply formula above
Orientation
det(M_{E,F}) > 0 ⟹ same orientation det(M_{E,F}) < 0 ⟹ opposite orientation Right-hand rule in E³ fixes orientation

Chapter 2 — Lines, Planes, Hyperplanes

Equations of Lines & Planes

Line in Aⁿ — parametric (base point Q, direction v)
OP = OQ + t·v, t ∈ ℝ i.e. xᵢ = qᵢ + t·vᵢ for each i
Line in Aⁿ — symmetric / Cartesian
(x₁−q₁)/v₁ = (x₂−q₂)/v₂ = ··· = (xₙ−qₙ)/vₙ (n−1 linear equations = intersection of n−1 hyperplanes)
Plane in A³ — parametric
OP = OQ + s·v + t·w, s,t ∈ ℝ
Plane in A³ — Cartesian (normal n=(a,b,c))
ax + by + cz + d = 0 Intercept form: x/α + y/β + z/γ = 1
Hyperplane in Aⁿ
H : a₁x₁ + ··· + aₙxₙ + aₙ₊₁ = 0 Normal vector: a = (a₁,…,aₙ) Direction space D(H): a₁x₁+···+aₙxₙ = 0

Relative Positions

Two lines in A² (solve system)
Unique point: rank = 2 (intersect) No solution: rank < rank(extended) (parallel) ∞ solutions: ℓ₁ = ℓ₂
Two planes in A³
Intersect in line: rank(M) = rank(M̃) = 2 Parallel: rank(M) < rank(M̃) Equal: rank(M) = rank(M̃) = 1
Two lines in A³ (via parametric)
Parallel: direction vectors v ∥ u Intersect: not parallel AND |x₁−x₂ y₁−y₂ z₁−z₂| |vx vy vz | = 0 |ux uy uz | Skew: not parallel, det ≠ 0 (no common point)
Line ℓ and plane π in A³
avx+bvy+cvz = 0 and P₀∉π → parallel (no meet) avx+bvy+cvz = 0 and P₀∈π → ℓ ⊆ π avx+bvy+cvz ≠ 0 → unique intersection t₀ = −(ax₀+by₀+cz₀+d)/(avx+bvy+cvz)

Chapter 3 — Euclidean Space, Scalar Product, Distance

Scalar Product & Angles

Scalar product (orthonormal basis)
⟨a, b⟩ = a₁b₁ + ··· + aₙbₙ = aᵀ·b |a| = √⟨a,a⟩ = √(a₁²+···+aₙ²) cos∡(a,b) = ⟨a,b⟩ / (|a|·|b|) a ⊥ b ⟺ ⟨a,b⟩ = 0
Orthogonal projection of b onto a
Pr⊥_a(b) = (⟨a,b⟩/⟨a,a⟩) · a = (⟨a,b⟩/|a|²) · a
Orthonormal frame: B = (e₁,…,eₙ), ⟨eᵢ,eⱼ⟩=δᵢⱼ
Normal vector to hyperplane H: a₁x₁+···+aₙxₙ=b
n = (a₁,…,aₙ) is normal to H P ∈ H ⟺ ⟨n, QP⟩ = 0 for any Q ∈ H
Angles between objects (orthonormal frame)
2 lines (dir. v₁,v₂): cos θ = ⟨v₁,v₂⟩/(|v₁||v₂|) 2 planes (norm. n₁,n₂): cos θ = ⟨n₁,n₂⟩/(|n₁||n₂|) Line–plane (dir v, norm n): acute angle = π/2 − arccos(⟨v,n⟩/(|v||n|))
Gram-Schmidt (orthonormalize basis {e₁,…,eₙ})
e′₁ = e₁ e′₂ = e₂ − Pr⊥_{e′₁}(e₂) e′₃ = e₃ − Pr⊥_{e′₁}(e₃) − Pr⊥_{e′₂}(e₃) Then normalize: ê_i = e′_i / |e′_i|

Distances

Distance between two points
d(P,Q) = |PQ| = √((p₁−q₁)²+···+(pₙ−qₙ)²)
Distance: point P to hyperplane H: aᵀx = b
d(P,H) = |a₁p₁+···+aₙpₙ − b| / √(a₁²+···+aₙ²) = |aᵀP + aₙ₊₁| / |a| (where H: aᵀx + aₙ₊₁ = 0)
Distance: point P to line ℓ in E³ (dir. v, through A)
d(P, ℓ) = |AP × v| / |v|
Distance: two parallel lines ℓ ∥ ℓ′ (B on ℓ′)
d(ℓ, ℓ′) = |AB × v| / |v|
Distance: two skew lines (A∈ℓ, B∈ℓ′, dir. v, w)
d(ℓ, ℓ′) = |⟨v×w, AB⟩| / |v×w| = |[v, w, AB]| / |v×w|
Locus at distance c from S (c > 0)
From a point → sphere of radius c From a line (E³) → cylinder of radius c From a hyperplane → two parallel hyperplanes
Locus equidistant from two objects
2 points → perpendicular bisecting hyperplane point + hyperplane → paraboloid (or parabola in E²) 2 hyperplanes → angle bisecting planes

Chapter 4 — Area, Cross Product & Volume

Area & Oriented Area (E²)

Area of parallelogram ABCD (v=AB, w=AD)
Area(ABCD) = |vx·wy − vy·wx| = |det[v w]| = |xA yA 1| |xB yB 1| |xD yD 1|
Oriented area (box product in E²)
[v, w] = vx·wy − vy·wx = det([v | w]) Areaₒᵣ(ABCD) = [v, w] Areaₒᵣ(△ABC) = [v, w] / 2 sin∡ₒᵣ(v,w) = [v,w] / (|v|·|w|)
Operator J (rotation by +90° in E²)
J(v) ⊥ v, |J(v)| = |v|, (v, J(v)) right-oriented J(vx, vy) = (−vy, vx) sin∡ₒᵣ(a,b) = pr⊥_{J(a)}(b) / |b|

Cross Product & Volume (E³)

Cross product a × b (right-hand rule)
a × b = | i j k | | a₁ a₂ a₃ | | b₁ b₂ b₃ | = (a₂b₃−a₃b₂, a₃b₁−a₁b₃, a₁b₂−a₂b₁) |a×b| = |a||b|sin∡(a,b) = Area(parallelogram) a×b ⊥ a, a×b ⊥ b, a ∥ b ⟺ a×b = 0 a×b = −b×a (skew-symmetric) Bilinear: (αu+βv)×w = α(u×w)+β(v×w)
Double cross formula
(a×b)×c = ⟨a,c⟩b − ⟨b,c⟩a
Box product [a,b,c] = ⟨a×b, c⟩ (mixed product)
[a,b,c] = |a₁ a₂ a₃| |b₁ b₂ b₃| |c₁ c₂ c₃| Vol(parallelepiped) = |[a,b,c]| Vol(tetrahedron ABCD) = |[AB,AC,AD]| / 6 [a,b,c]=0 ⟺ a,b,c coplanar (linearly dependent) [a,b,c] > 0 ⟺ (a,b,c) right-oriented
Euler-Rodrigues (rotation by θ around unit axis v)
Rot_{v,θ}(x) = cos(θ)x + sin(θ)(v×x) + (1−cosθ)⟨v,x⟩v

Volume & Common Perpendicular

Volume formulas
Rectangle a×b: Area = ab Parallelogram (v,w): Area = |v×w| ... wait, this is E³ Triangle (base a, h): Area = ah/2 Parallelepiped (a,b,c): Vol = |[a,b,c]| Pyramid (base A, h): Vol = Ah/3 Tetrahedron ABCD: Vol = |[AB,AC,AD]|/6
Common perpendicular of two skew lines ℓ,ℓ′ (A∈ℓ, B∈ℓ′, dir. v, w)
Direction of perp: a = v × w Normal of π (contains ℓ): n = (v×w)×v Normal of π′ (contains ℓ′): n′ = (v×w)×w π: |x−xA y−yA z−zA| |vx vy vz | = 0 |ax ay az | π′: |x−xB y−yB z−zB| |wx wy wz | = 0 |ax ay az | Common perp d = π ∩ π′

Chapter 5 — Affine Maps, Projections & Reflections

Affine Maps & Homogeneous Coordinates

Affine map φ: φ(P) = A·[P]_K + b

lin(φ) has matrix A; translation b. Invertible ⟺ A invertible.

Homogeneous matrix
M̂(φ) = [A | b] (n+1)×(n+1) matrix [0 | 1] Composition: M̂(ψ∘φ) = M̂(ψ)·M̂(φ)
Tensor (outer) product
v ⊗ a = v·aᵀ (n×n matrix, (v⊗a)ᵢⱼ = vᵢaⱼ) (v⊗a)·w = ⟨a,w⟩·v (orthonormal basis) (v⊗a)ᵀ = a⊗v

Proj ∈ End_aff but ∉ AGL (not bijective).

Reflections ∈ AGL (bijective).

Key relation: Ref = 2·Pr − Id

Projection & Reflection on Hyperplane H

H: a₁x₁+···+aₙxₙ+aₙ₊₁=0, direction v (not ∥ H)

Projection onto H parallel to v
Pr_{H,v}(P) = (Iₙ − v·aᵀ/(vᵀ·a))·P − (aₙ₊₁/(vᵀ·a))·v lin part: Iₙ − (v⊗a)/⟨v,a⟩
Reflection in H parallel to v
Ref_{H,v}(P) = (Iₙ − 2v·aᵀ/(vᵀ·a))·P − (2aₙ₊₁/(vᵀ·a))·v lin part: Iₙ − 2(v⊗a)/⟨v,a⟩
Orthogonal versions (v = a = normal)
Pr⊥_H(P) = (Iₙ − a⊗a/|a|²)·P − (aₙ₊₁/|a|²)·a Ref⊥_H(P) = (Iₙ − 2a⊗a/|a|²)·P − (2aₙ₊₁/|a|²)·a

Projection & Reflection on Line ℓ

ℓ: dir. v, through Q. W = direction space of projection (W: aᵀx=0, W not ∥ ℓ)

Projection onto ℓ parallel to W
Pr_{ℓ,W}(P) = (v·aᵀ/(vᵀ·a))·P + (Iₙ−v·aᵀ/(vᵀ·a))·Q lin part: (v⊗a)/⟨v,a⟩
Reflection in ℓ parallel to W
Ref_{ℓ,W}(P) = (2v·aᵀ/(vᵀ·a)−Iₙ)·P + 2(Iₙ−v·aᵀ/(vᵀ·a))·Q lin part: 2(v⊗a)/⟨v,a⟩ − Iₙ
Orthogonal versions (W ⊥ ℓ, so v=a=dir. of ℓ)
Pr⊥_ℓ(P) = (a⊗a/|a|²)·P + (Iₙ−a⊗a/|a|²)·Q Ref⊥_ℓ(P) = (2a⊗a/|a|²−Iₙ)·P + 2(Iₙ−a⊗a/|a|²)·Q

How to Apply (Method II)

Given H: x+y−1=0 and v(2,1):

Step 1: identify a, aₙ₊₁, v
a = (1,1), aₙ₊₁ = −1, v = (2,1)
Step 2: compute vᵀ·a (scalar)
vᵀ·a = 2·1 + 1·1 = 3
Step 3: compute v·aᵀ (matrix)
v·aᵀ = [2]·[1 1] = [2 2] [1] [1 1]
Step 4: plug into formula
Pr = (I − (1/3)[2 2])·P − (−1/3)·[2] [1 1] [1] = (1/3)[1 −2]·P + (1/3)[2] [−1 2] [1]

Chapter 6 — Isometries

Affine Form & Orthogonal Matrices

Isometry: d(φ(P),φ(Q))=d(P,Q). Every isometry is affine: φ(x)=Ax+b.

Characterization
φ isometry ⟺ A⁻¹=Aᵀ ⟺ AᵀA=Iₙ (A orthogonal) det(A)=±1 det(A)=+1: direct isometry (displacement) det(A)=−1: indirect isometry
Rotation matrix SO(2) — all elements
Rθ = [cosθ −sinθ] det=+1 [sinθ cosθ] cos θ = tr(lin(φ)) / 2
Rotation around point C(c₁,c₂) in E²
Homogeneous matrix = T_C · Rθ · T_{−C} [cosθ −sinθ −c₁cosθ+c₂sinθ+c₁] [sinθ cosθ −c₁sinθ−c₂cosθ+c₂] [0 0 1 ]

Classification in E²

Chasles' theorem — direct (det A = +1)
A = I₂, b = 0 → identity A = I₂, b ≠ 0 → translation by b A = Rθ, θ ≠ 0 → rotation (unique fixed point p): b = (I₂−Rθ)p cos θ = tr(A)/2
Indirect (det A = −1)
reflection in line ℓ (if no translation along ℓ) glide-reflection in ℓ (= reflection + translation ∥ ℓ) Lemma: Rθ · Ref⊥_{Ox} = reflection in y=tan(θ/2)·x

Rotations in E³

Euler's theorem — rotation around axis Rv
Every A ∈ SO(3) fixes a line (det(A−I₃)=0) cos θ = (tr(lin(φ)) − 1) / 2
Euler-Rodrigues formula (unit axis v)
Rot_{v,θ}(x)=cos(θ)x+sin(θ)(v×x)+(1−cosθ)⟨v,x⟩v
Rotation matrices around axes
Rotx,θ=[1 0 0 ] Roty,θ=[cosθ 0 sinθ] [0 cosθ −sinθ] [0 1 0 ] [0 sinθ cosθ] [−sinθ 0 cosθ] Rotz,θ=[cosθ −sinθ 0] [sinθ cosθ 0] [0 0 1]
Euler angles: any SO(3) matrix = Rotz,γ · Rotx,β · Rotz,α
α,β,γ ∈ [0,2π)×[0,π)×[0,2π) → unique rotation

Classification in E³ & Surfaces of Revolution

Chasles — direct (det A = +1)
identity | translation | rotation | glide-rotation (glide-rotation = rotation + translation ∥ axis)
Indirect (det A = −1)
reflection in plane π glide-reflection in π (+ translation ∥ π) rotation-reflection (+ rotation ⊥ π)
Surface of revolution
Rotate a curve around axis ℓ using Rotₓ,β (or similar): Sphere: rotate circle (cosα, sinα, 0) around x-axis → (cosα, cosβsinα, sinβsinα) Cylinder: translate line, or rotate ∥ line Hyperboloid: rotate skew line around z-axis

Chapter 9 — Classification of Conics (Isometric & Invariants)

Setup & Reduction Algorithm

General conic: C: q₁₁x²+2q₁₂xy+q₂₂y²+2b₁x+2b₂y+c=0

Matrices
Q = [q₁₁ q₁₂] (symmetric matrix of quadratic part) [q₁₂ q₂₂] Q̂ = [q₁₁ q₁₂ b₁] (extended symmetric matrix) [q₁₂ q₂₂ b₂] [b₁ b₂ c ]
Algorithm: reduce to canonical form
Step 1 – Rotation: find eigenvalues λ₁,λ₂ of Q eigenvectors → orthonormal basis B′ ∈ SO(2) equation becomes: λ₁x′²+λ₂y′²+v₁x′+v₂y′+c=0 Step 2 – Translation: complete the square for λᵢ≠0 → λ₁x″²+λ₂y″²=k or λ₁x″²+v₂y″=k Step 3 – (optional) swap axes to get canonical form
Computing M_{B,B′}: columns = eigenvectors in B
Ensure det(M_{B,B′}) = +1 (flip sign of one eigenvector if needed) New eq: λ₁x′²+λ₂y′²+bM_{B,B′}[x′,y′]ᵀ+c = 0
